

## 📝 Funcionalidades Principales

*   **Panel de Administrador:** Creación, edición y cierre de foros temáticos.
*   **Módulo de Discusión:** Los ciudadanos pueden iniciar hilos de conversación dentro de los foros creados.
*   **Sistema de Comentarios:** Interacción multinivel en cada discusión.


# Santo Domingo Conecta - Plataforma de Participación Ciudadana

![Banner del Proyecto](https://github.com/user-attachments/assets/83bd1697-cfbf-48f4-a241-334039957f14)

**Santo Domingo Conecta** es una plataforma web diseñada para fortalecer el vínculo entre la municipalidad y sus ciudadanos. Inspirada en la estructura de foros dinámicos, permite la creación de espacios de discusión moderados, permitiendo que la comunidad participe activamente en la toma de decisiones y el diálogo local.

## 🚀 Tecnologías Utilizadas

El proyecto emplea un stack moderno para garantizar escalabilidad y una experiencia de usuario fluida:

*   **Frontend:** React.js (Vite) + Tailwind CSS (v4) + TypeScript.
*   **Backend:** Node.js + Express.
*   **Base de Datos:** MySQL.
*   **Gestión de Monorepo:** npm workspaces.

## 🛠️ Requisitos e Instalación

### Requisitos previos
*   Node.js (versión 18 o superior)
*   npm (versión 9 o superior)

### Configuración del Repositorio
1.  Clona el repositorio:
    ```bash
    git clone https://github.com/Bresman-vargas/proyecto_publico.git
    cd proyecto_publico
    ```
2.  Instala las dependencias de todo el proyecto:
    ```bash
    npm install
    ```
